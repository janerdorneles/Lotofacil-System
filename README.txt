# Conferência Automática Lotofácil

Conferidor visual e inteligente para jogos da Lotofácil.

## Recursos
- Seleção visual de dezenas
- 4 jogos simultâneos
- Conferência automática
- Destaque para prêmios
- Interface responsiva
- 100% offline

## Como usar
1. Selecione as 15 dezenas sorteadas
2. Marque suas apostas nos Jogos 1–4
3. Clique em "Conferir Jogos"
